# Task Tracker MVP

## Описание
Это минимально жизнеспособный продукт (MVP) — локальный трекер задач, разработанный для демонстрации ключевой ценности: простого управления задачами без серверов или регистрации. Приложение позволяет добавлять, 
редактировать, удалять задачи, изменять их статус и приоритет. Данные хранятся в LocalStorage браузера.

Технологии: HTML5, CSS3, vanilla JavaScript.

## Требования
- Современный браузер (Google Chrome, Mozilla Firefox или аналогичный).
- Нет дополнительных зависимостей или серверов — работает полностью локально.
